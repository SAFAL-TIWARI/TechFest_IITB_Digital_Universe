# TechFest IITB Digital Universe

A fully responsive 3D interactive website created for the Techfest IIT Bombay Campus Ambassador website creation task.

The website presents **"The Techfest Digital Universe"**, where users scroll through a futuristic technology world featuring a particle galaxy, holographic sphere, digital timeline, tech domains, metaverse city, robot assistant, live stats, innovation cube, and immersive journey animations.

## Tech Stack

- HTML5
- CSS3
- JavaScript
- Three.js
- GSAP
- ScrollTrigger

## Features

- Fully responsive 3D landing page
- Animated particle galaxy background
- Floating holographic Techfest sphere
- Scroll-based 3D transitions
- Interactive tech domain cards with modal popups
- Futuristic metaverse city scene
- Interactive robot assistant
- Animated live statistics counters
- Rotating innovation challenge cube
- Glassmorphism and neon cyber-tech UI

## How To Run

Open `index.html` directly in a browser.

For best results, use a local server such as the VS Code Live Server extension.

## Project Purpose

This project was built as part of my **Campus Ambassador journey for Techfest IIT Bombay**, showcasing creativity, frontend development, 3D web interaction, and modern animation techniques.
